# 🃏 Egyptian Jacks

A modern multiplayer card game built with a scalable monorepo architecture.

## 🚀 Tech Stack

* Frontend: React + Vite + TypeScript
* Backend: Node.js (planned)
* Realtime: Socket.io (planned)
* Game Engine: Custom logic (in progress)

## 📂 Project Structure

```
apps/
  web/        # Frontend (React)
  server/     # Backend (API + sockets)

packages/
  game-engine/    # Core game logic
  shared-types/   # Shared TypeScript types
```

## 🛠️ Getting Started

### Install dependencies

```bash
pnpm install
```

### Run frontend

```bash
pnpm dev:web
```

## 🌐 Local Development
